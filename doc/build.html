Build guide

  • Windows
  • Linux

    • Windows

    You need wsl or emulator to use grub!

    Install:

  • i686-elf-tools for windows
  • Fasm compiler
  • Install Ubuntu on WSL
  • Last stable source code of SynapseOS

    • Install grub using WSL:

  • sudo apt-get update
  • sudo apt-get install grub-common
  • sudo apt-get install xorriso
  • sudo apt-get install grub-pc-bin

    • Compile

  • Add Fasm and i686-elf-tools to %PATH%
  • Run build.bat

    • linux

    Install:

  • i686-elf-tools for Linux
  • Fasm compiler
  • Last stable source code of SynapseOS

    • Install grub and fasm:

  • sudo apt-get update
  • sudo apt-get install grub-common
  • sudo apt-get install fasm
  • sudo apt-get install xorriso
  • sudo apt-get install grub-pc-bin

    • Compile

  • Add Fasm and i686-elf-tools to %PATH%
  • Run build.sh in the "/ src / kernel" folder
